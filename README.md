# HR Attrition Analysis Report 📊

This project presents an interactive HR Attrition Analysis dashboard developed using Power BI. The goal is to analyze employee data and identify key factors contributing to employee attrition within an organization.

## 📌 Objective
To understand patterns of employee attrition and provide insights that can help organizations improve employee retention and workforce management.

## 🛠️ Tools & Technologies
- Power BI
- Data Cleaning & Transformation (Power Query)
- Data Modeling
- DAX (Data Analysis Expressions)

## 📊 Key Features
- Interactive dashboard with filters and slicers
- Visualization of attrition trends across different departments
- Analysis based on:
  - Age groups
  - Gender
  - Job roles
  - Salary slabs
  - Years at company
- KPI indicators such as:
  - Total Employees
  - Attrition Count
  - Attrition Rate
  - Average Age & Salary

## 📈 Key Insights
- Higher attrition observed among employees with lower salary ranges
- Employees with fewer years at the company showed higher attrition rates
- Certain job roles experienced more attrition compared to others
- Age group 25–35 showed significant attrition trends

## 📁 Project Files
- `HR_Attrition_Report.pbix` – Power BI dashboard file
- `dataset.csv / excel` – Source dataset
- `README.md` – Project documentation

## 🚀 Conclusion
The dashboard provides actionable insights into employee attrition patterns, helping organizations make data-driven decisions to improve employee satisfaction and reduce turnover.
